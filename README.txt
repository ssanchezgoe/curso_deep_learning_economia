Curso de Deep Learning en Economía

- Metodología: El curso se dividirá en sesiones teórico prácticas a través de notebooks (NB) implementados en python (google colab y databricks)

- Requisitos: Tener una cuenta de gmail y una cuenta de databricks community (es gratis).

  Nota: Para instalaciones locales de librerías, se realizará a través de anacondo, pero no se dará soporte a estas instrucciones.
        Las instalaciones locales de librerías deben ser asumidas de forma autónoma por el estudiante.

- Estructura de carpetas:

    * NBs_Google_Colab: Carpeta que almacena los notebooks de las sesiones en Google Colab
    * NBs_Databricks: Carpeta que almacena los notebooks algunos notebooks de las sesiones en databricks.

    Nota: Todas las carpetas cuentan con un README que describe el contenido almacenado en dicha carpeta.

- Secciones:

Cada sesión estará compuesto por un notebook cuyo nombre seguirá la nomenclatura

DL_S[Numero]_[Nombre_De_la_Temática]

y serán almacenados todos, sin excepción, en la carpeta NBs_Google_Colab. Algunos notebooks se almacenarán en la carpeta NBs_Databricks para demostrar funcionalidades
propias de databricks.

- Notebooks

1. DL_S00_NB_Intro.ipynb: Notebook introductorio de empleo de notebooks en colab y databricks.
2. DL_S01_Data_Manipulation: Noteboook de repaso de manipulación de datos en python.
3. DL_S03_Time_Series: Notebook de manipulación de series de tiempo.
4. DL_S04_ML_Review: Notebook de repaso de algoritmos de Machine Learning.
5. DL_S05_DL_Intro: Notebook introductorio a deep learning.
